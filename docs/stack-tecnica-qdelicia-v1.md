# Stack Técnica — QDelícia Frutas

**Decisão final. Sem alternativas.**

---

## Framework
- **Next.js 15** — App Router, `output: 'export'` (site estático)

## Linguagem
- **TypeScript**

## Estilização
- **Tailwind CSS v3**

## Animações
- **Framer Motion** — expansão do painel de variações dos cards
- **CSS transitions puras** — hover dos cards (scale, shadow, fade do label CTA)

## Ícones
- **Lucide React**

## Imagens
- **next/image** — WebP automático, lazy loading, srcset responsivo

## Formulário
- **React Hook Form** — validação e controle dos campos
- **Resend** — envio do e-mail via API Route do Next.js

## Utilitários
- **clsx** — composição condicional de classes
- **tailwind-merge** — merge seguro de classes Tailwind

## Embed de vídeo
- **react-lite-youtube-embed** — carrega o player do YouTube só no clique
